Plug-Ins. Plug-Outs. What?

Annotorious is a flexible system that can be extended in all sorts of ways. Extensions to Annotorious are called plug-ins, and they embed additional functionality into Annotorious such as tagging, or the ability to connect to different storage backends.

Ok. Now here comes the complicated part: Annotorious was also designed to be itself embeddable into other systems. Components that plug Annotorious into another tool or environment (rather than the other way round) are called (for lack of a better word) plug-outs. For example, there is a plug-out which seamlessly integrates Annotorious into the Annotator Web annotation system by the Open Knowledge Foundation.

Plug-Ins

To add a plugin, include the plugin script in your page head and register the plugin through the Annotorious JavaScript API. Example: 

<head>
  <!-- Import main Annotorious script and CSS -->
  <link rel="stylesheet" href="css/annotorious.css" type="text/css" />
  <script src="js/annotorious.min.js"></script>

  <!-- Import the plugin script -->
  <script src="js/example-plugin.js"></script>
  ...
  <!-- Use the API to attach and activate the plugin -->
  <script>
    anno.addPlugin('ExamplePlugin', {});
  </script>
</head>
